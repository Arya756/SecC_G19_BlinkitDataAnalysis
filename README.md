# DVA_Capstone2_C_G19

## Project Overview
This project focuses on analyzing the Blinkit dataset to uncover insights through data visualization and analytics.

## Folder Structure
- `data/`: Contains project data.
  - `raw/`: Unmodified data (e.g., `blinkit_dataset.csv`).
  - `processed/`: Cleaned and transformed data for analysis.
- `notebooks/`: Jupyter notebooks used for data processing.
  - `01_extraction.ipynb`: Initial basic exploratory data analysis (EDA).
  - `02_cleaning.ipynb`: Data cleaning operations.
- `docs/`: Project documentation.
- `reports/`: Generated reports.
- `scripts/`: Custom Python scripts/modules.
- `tableau/`: Tableau workbooks for dashboards and visualizations.

## Progress So Far
* **Repository Setup:** Project structure established with dedicated directories for raw/processed data, notebooks, and scripts. Collaborative Git environment set up (`main` and `dev` branches).
* **Data Extraction & Initial EDA:** 
  * Loaded the raw `blinkit_dataset.csv` into a Pandas Dataframe.
  * Extracted shape characteristics and column attributes.
  * Generated reports on missing values within the dataset.
  * Extracted basic descriptive statistics for initial evaluation.
